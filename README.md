Mathematical Expression Parser
A lightweight, modular expression parser built from scratch in C++. This project serves as a deep dive into algorithmic logic, string manipulation, and system-level software engineering, with the goal of evolving into a high-performance evaluation engine.

Current Status: Phase 1 (Unit Digits)
The parser currently implements core logic to extract, validate, and compute single-unit integers. This phase focused on establishing a robust foundation for character processing and error handling without external dependencies.

Roadmap
The parser is currently in active development. The planned trajectory for future updates is as follows:

[x] Phase 1: Foundation (Unit Digit extraction & basic validation).

[ ] Phase 2: Multi-digit and decimal support.

[ ] Phase 3: Operator Precedence implementation using the Shunting Yard Algorithm.

[ ] Phase 4: Abstract Syntax Tree (AST) construction for complex expression evaluation.
